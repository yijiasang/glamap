## Project Overview
Glamap is a location-based web application that connects freelance beauty service providers with clients across Australia. The platform addresses visibility challenges faced by independent beauty professionals while giving clients an intuitive, map-driven way to discover nearby, trustworthy providers.

This project was built end-to-end as a personal portfolio project to demonstrate modern full-stack development, secure authentication, and scalable system design.

---

## Key Features
- **Interactive Map Discovery**  
  Leaflet.js-powered map with custom markers covering 160+ Sydney suburbs and provider locations
- **Advanced Search & Filtering**  
  Location-based directory search by suburb, postcode, and service type
- **Provider Profiles**  
  Detailed profiles including services, pricing, ratings, and client reviews
- **Review System**  
  One-review-per-client policy with optional display names to ensure authenticity
- **Real-Time Messaging**  
  Private client-provider messaging with notifications
- **Profile Management**  
  Secure image upload, edit, and removal using object storage

## Tech Stack
**Frontend**
- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- TanStack React Query
- Leaflet.js
- React Hook Form + Zod

**Backend**
- Node.js + Express.js (TypeScript REST API)
- PostgreSQL
- Drizzle ORM

**Authentication & Infrastructure**
- OpenID Connect (OIDC)
- Object storage for media uploads
- OpenStreetMap integration
- Deployed with Vercel (frontend) & Render (backend)
